# CTF Handbook

A personal knowledge base for collecting and organizing techniques, tools, and notes encountered during CTF competitions and security research.

View rendered pages at [chhu0830.github.io/ctf](https://chhu0830.github.io/ctf).

## Coverage

```
docs/
├── exploit/
│   ├── binary/         # Binary exploitation, ELF/PE analysis, debuggers, static/dynamic analysis
│   ├── network/        # HTTP, Wi-Fi, OS exploitation
│   │   └── web/        # Web security, scanners, payloads, OWASP references
│   └── system/         # Linux, Windows, macOS privilege escalation and forensics
└── auxiliary/
    ├── crypto/         # Cryptanalysis and encoding
    ├── payload/        # Payload collections
    ├── recon/          # OSINT, DNS, discovery
    └── misc/           # QR code, steganography, and other misc techniques
```

## Contributing

Contributions are welcome! If you have techniques, tools, or notes worth sharing, feel free to open a PR.

- Follow the existing directory structure and place content under the appropriate category
- Keep notes concise — commands, tables, and brief explanations are preferred over lengthy prose
- Include sources where possible

## Disclaimer

Some content in this repository is adapted from course materials, CTF writeups, and publicly available online resources.

If you believe any content originates from your work and is missing proper attribution, please open an issue — I will add the source as soon as possible.

This repository is intended for educational purposes only.
